 LingkodHub

LingkodHub is a web application that helps barangay residents easily find and hire trusted local service providers. Inspired by how communities currently post requests on social media, this platform streamlines the process by showing volunteer profiles, skills, ratings, and location, allowing users to select the best fit for their tasks.

## Features
- User registration and login
- Post and view job requests
- Browse worker profiles with ratings and skills
- Accept and manage job applications
- Localized experience for barangay communities

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js (Express) / Python (Flask)
- Database: MySQL / SQLite
- Containerization: Docker
- Deployment: Kubernetes (local cluster)

## Project Goals
- Simulate a real-world application with DBMS, API, and frontend
- Introduce containerization and orchestration using Docker and Kubernetes
- Demonstrate scalability, self-healing, and version control in deployment
